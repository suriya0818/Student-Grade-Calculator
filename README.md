Student Grade Calculator

This project helped me understand how decision-making works in Python using if-elif-else statements. I also learned how to use loops for input validation and functions to organize code better. Through this project, I gained more confidence in writing Python programs that interact with users and produce different outputs based on the given input.



Project Overview



The Student Grade Calculator is a simple Python program that accepts a student's name and marks, calculates the grade based on the marks entered, and displays an encouraging message. The program also checks whether the entered marks are valid and asks the user to enter them again if they are outside the range of 0 to 100.



Setup Instructions



1\. Install Python on your computer.

2\. Save the program file as grade\_calculator.py.

3\. Open Command Prompt or Terminal.

4\. Go to the folder where the file is saved.

5\. Run the program using:



python grade\_calculator.py

Technical Details



The program uses:



\* Variables to store user input.

\* A function to calculate the grade and display the appropriate message.

\* if-elif-else statements for grading logic.

\* A while loop to validate user input.

\* print() statements to display results.



Program Flow:



User enters name and marks → Marks are validated → Grade is calculated → Result and message are displayed
